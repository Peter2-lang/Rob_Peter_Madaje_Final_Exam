# Student Registration System (Final Exam)

This is a full-stack CRUD application built with Node.js, Express, and PostgreSQL, deployed on Render with an Aiven cloud database.

## Features
- **Create**: Register a new student.
- **Read**: View all students in a data table.
- **Update**: Edit existing student records.
- **Delete**: Remove students from the database.

## Tech Stack
- Frontend: HTML5, CSS3 (Bootstrap 5), Vanilla JavaScript
- Backend: Node.js, Express.js
- Database: final

## Deployment Link
- Live Site: `https://rob-peter-madaje-final-exam.onrender.com/`

## Setup / Installation (Local)
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file in the root directory and add your Aiven DB connection string: `DATABASE_URL=postgres://user:password@host:port/dbname?sslmode=require`
4. Run `npm start`.
5. Access the app at `http://localhost:3000`.
